## 使用说明

直接使用NotebookLM生成Infographic的时候, 如果不加任何提示词, 风格就会不稳定, 所以需要加上提示词让风格稳定.

这个提示词是书房后院的一个院友发的图, 我看着好看, 就问了大模型总结了风格, 提示词如下:

```
“手绘插画、内容详尽的海报风格、米色纸张背景、醒目的图标、清晰有序的布局”
```

生成Infographic的时候, 直接写上这个提示词就行.
