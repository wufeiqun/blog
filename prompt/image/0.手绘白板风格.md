## 手绘白板风格

适合一些知识拆解场景, 清晰简洁.

## 提示词

1. 想了解一个知识点, 直接让大模型画图

```
请生成一张手绘风白板图，主题是「XXX」。

要求：
- 手绘白板风格，像老师在白板上讲课
- 用彩色马克笔风格画出示意箭头等
- 以可读文本标注关键概念
- 使用中文注释
- 9:16, 4K
```

2. 自己有一些文本内容想让大模型给画图

```
XXX文本内容XXX

基于以上内容帮我生成一张手绘风白板图。

要求：
- 手绘白板风格，像老师在白板上讲课
```
