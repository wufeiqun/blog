#### 背景介绍

公司有一个项目启动了两个后端的服务, 想使用一个域名代理, 类似如下:

```
server {

  location /v1 {
    proxy_pass http://one;
  }

  location /v2 {
    proxy_pass http://two;
  }
}
```

如果按照这个配置的话, 请求到后端的请求都会自动加上`/v1`或者`/v2`, 后端想要去掉这个, 也就是如下:

```
/v1/api/list   --到后端-->  /api/list
```

#### 解决方式

nginx配置如下:

```
server{

      location /v1 {
          rewrite ^/v1/(.*)$ /$1 break;
          proxy_pass http://one;
      }

}
```
